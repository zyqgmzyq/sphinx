## [1. leetcode32 - 最长有效括号](https://leetcode-cn.com/problems/longest-valid-parentheses)   ***

### 题目描述

给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

 

输入：s = "(()"
输出：2



### 思路分析

### 代码

```c++
class Solution {
public:
    int longestValidParentheses(string s) {
        int ans = 0;
        int left = 0;
        int right = 0;
        for (int i = 0; i < s.size(); i++) {
            if (s[i] == '(') {
                left++;
            } else {
                right++;
            }
            if (left == right) {
                ans = max(ans, left * 2);
            } else if (right > left) {
                left = 0;
                right = 0;
            }
        }

        left = 0;
        right = 0;
        for (int i = s.size() - 1; i >= 0; i--) {
            if (s[i] == '(') {
                left++;
            } else {
                right++;
            }
            if (left == right) {
                ans = max(ans, left * 2);
            } else if (left > right) {
                left = 0;
                right = 0;
            }
        }


        return ans;
    }
};
```
